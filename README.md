# ConnectFour
Deze repository is aangemaakt voor de afstudeerstage bij Alten voor de Master HCAI. De opdracht voor het afsturen is het maken van een addaptive AI voor het spel vier op een rij. De repository bevat een zelf geschreven digitale versie van vier op een rij en hiernaast ook de AI. De code is aan te roepen door "main.py" te runnen.
